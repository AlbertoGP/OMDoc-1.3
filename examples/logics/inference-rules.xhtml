Conceptschema
This is the binding symbols for schematic inference rules. All the bound variables may be freely instantiated.
Conceptrule
An inference rule is an n+1-ary relation on judgments. The first n arguments are called the antecedents, and the last one the consequent. We read an inference rule with the intuition that if the antecedents have been derived, then the consequent can be.
