Classical Truth Values
Michael Kohlhase
2002-02-27
This theory provides all classical two-valued propositional connectives, it shares the symbol names with the OpenMath content dictionary logic1 for consistency. We have a separate theory here, since we want to make the semantics more restrictive than the inclusive treatment in logic1. In particular, we insist that the connectives are two-valued.
