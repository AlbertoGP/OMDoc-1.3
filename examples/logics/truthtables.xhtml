Truth Tables.
Michael Kohlhase
2002-04-11
This theory generalizes truth tables.
Concepttt
The truth table constructor for a binary (n x m) operator. It takes n + 2 arguments. The first one is the symbol for the defined operator. Then come n values of the second argument (the first row), and finally n rows using the ttrow symbols.
Conceptttrow
A row in a (n x m) truth table, the first argument is the first argument, and the rest of the arguments are the m values
Conceptttargs
The vector of second arguments for a binary truth tables.
