Strong First-Order Kleene Logic
Michael Kohlhase
2002-06-12
This theory provides the three-valued first-order quantifiers for strong Kleene Logic.
Conceptforall
This symbol is the binding symbol for the three-calued universal ("for all") quantifier.
Example

All objects are equal to themselves

∀xx=x
Conceptexists
This symbol is the binding symbol for the three-valued existential ("there exists") quantifier.
Example

If the universe is non-empty, then the following statement is true

∃xx=x
