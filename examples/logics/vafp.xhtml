This document contains and defines the constants used in the Verifun to OMDoc translation.

Conceptfunction
This symbol is used to represent anonymous functions as lambda expansions. It is used in a binder that takes two further arguments, the first of which is a list of variables, and the second of which is an expression, and it forms the function which is the lambda extraction of the expression
Example

The following is a representation of the identity function:

λXX
Conceptlet
The unary let binder
Conceptlet-value
The attribute for the value of a variable under a let binding
Example

The following is an example for function constructed with the help of a let binding

plus=λX,Y if X=0 then Y else let in plus(X,Y)end fi
Conceptif
This symbol is the ternary if/then/else operator.
