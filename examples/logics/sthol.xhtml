This theory provides the classical two-valued first-order quantifiers.

Conceptforall
This symbol is the binding symbol for the universal quantifier.
Example

All objects are equal to themselves

∀xx=x
Conceptexists
This symbol is the binding symbol for the existential ("there exists") quantifier.
Example

If the universe is non-empty, then the following statement is true

∃xx=x

Some authors prefer to use higher-order abstract syntax for quantifiers, this variant only uses the binding symbol λ and models the quantifiers as higher-order predicates.

Conceptforall-op
This symbol is universal predicate of higher-order abstract syntax.
Example

All objects are equal to themselves

Π(λxx=x)
Conceptexists-op
The existential ("there exists") quantifier of higher-order abstract syntax.
Example

If the universe is non-empty, then the following statement is true

Σ(λxx=x)

The theory of higher-order logic with a choice operator.

Conceptchoose
This symbol is the binding symbol for choice operator.
Axiom (The axiom of choice for ℩.)

For any property P, that can be satisfied by some x, chooses a concrete witness.

∀P∃XP(x)⇒P(℩yS)
Conceptchoice-op
This symbol is a choice function.
Axiom (The axiom of choice for ι.)

For any non-empty set S, ι returns a member of S.

∀S∃XS(x)⇒S(℩(S))
