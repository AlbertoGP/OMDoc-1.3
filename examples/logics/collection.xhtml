A collection of OMDoc documents that define the most commonly used logics

1. Introduction

This theory collection makes available an interdependent collection of theories that define logics. These logics are a foundation for the interoperability and communication between reasoning services on MathWeb.

2. Propositional Logics

Propositional logics are logics that abstract from the inner structure of statements, and only study their interrelation in complex formulae built up from propositional connectives.

3. First-Order Logics

First-order logics are logics that allow quantification over individuals. They are usually untyped.

4. Untyped Calculi

Untyped lambda calculus and other calculi with built-in notions of equality

5. Type Systems

Type systems are basic building blocks of many logics.

6. Typed Logics and Type Theories

In this section we will introduce logics that use type systems to categorize the basic building blocks.
